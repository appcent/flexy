Breakpoint

Использование flexy breakpoint

Block grid представляет из себя аналог одноименного компонента из foundation

Настройки

$flexy-breakpoints: (
    small: 0,
    medium: 768px,
    large: 992px,
    xlarge: 1200px,
    xxlarge: 1600px
) !default;

`$flexy-breakpoints` это `map` из sass где в качестве ключа выступает любое удобное для вас название media

Пример:

$flexy-breakpoints: (
    foo: 100px,
    bar: 350px
);

.my-class {
    font-weight: normal;

    @include flexy-breakpoint(bar) {
        font-weight: bold;
        ...
    }
}

Block grid использует названия из настроек `flexy-breakpoint` в качестве модификаторов.

Внимание

Все блоки имеют по умолчанию media query от N и не фиксируются по верхнему значению:

@media screen and (min-width: 200px) {
    ...
}

Описание функций и миксинов

flexy-breakpoint

Принимает в качестве аргументов `(string, string)`

Example
Source
.my-class {
    @include flexy-breakpoint(bar, print) {
        color: red;
    }
}
@media print and (min-width: 350px) {
    .my-class {
        color: red;
    }
}

flexy-media

Принимает в качестве аргументов `(map, string)`

Example
Source
.my-class {
    @include flexy-media((min-width: 100px, max-width: 110px), print) {
        color: red;
    }
}
@media print and (min-width: 100px) and (max-width: 110px) {
    .my-class {
        color: red;
    }
}

flexy-get-breakpoint

Функция принимает в качестве аргументов `(string)`

Example
Source
flexy-get-breakpoint(foo);
350px

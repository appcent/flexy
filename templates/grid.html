Описание

Сетка с использованием flex. Для корректной работы требует flexy-breakpoint

Настройки

{% filter escape %}// Количество колонок
$flexy-grid-columns: 12 !default;

// Отступ между колонками
$flexy-grid-gutter: 0.5rem !default;

// Максимальная ширина контейнера
$flexy-grid-wrapper-width: 100% !default;

// Класс для строки
$flexy-grid-class-row: 'row' !default;

// Класс для контейнера (wrapper)
$flexy-grid-class-wrapper: 'wrapper' !default;

// Класс для колонки
$flexy-grid-class-col: 'col' !default;

// Класс для отступов
$flexy-grid-class-off: 'off' !default;
{% endfilter %}

Заголовки

Миксин для указания стилей заголовкам.

Результатом выполнения будет сгенерированные стили для `h1, h2, h3, h4, h5, h6` с применением размеров и стилей которые вы укажите.

Example
Source
@include flexy-heading {
    border-bottom: 1px solid silver;
}
h1 {
    font-size: 2em; // xxlarge
    border-bottom: 1px solid silver;
}
h2 {
    font-size: 1.5em; // xlarge
    border-bottom: 1px solid silver;
}
// ... etc

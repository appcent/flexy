Visibility

Оглавление

  1. Описание
  2. Использование
    1. Отображение (visible)
    2. Скрытие (hide)

Описание

Отображение и скрытие блоков в зависимости от flexy-breakpoint

Использование

Отображение

Отображать для small

Код:

{% filter escape %}{{ visible('visible', 'small', 'only') }}{% endfilter %}

Пример (изменяйте размер браузера):

{{ visible('visible', 'small', 'only') }}

Отображать для small и выше

Код:

{% filter escape %}{{ visible('visible', 'small', 'up') }}{% endfilter %}

Пример (изменяйте размер браузера):

{{ visible('visible', 'small', 'up') }}

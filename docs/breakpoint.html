Breakpoint

Миксины и функции созданы для облегчения работы с @media

Оглавление

  1. Настройки
  2. Использование
    1. flexy-breakpoint
    2. flexy-media

Настройки:

$flexy-breakpoints: (
        small: 0,
        medium: 768px,
        large: 992px,
        xlarge: 1200px,
        xxlarge: 1600px
) !default;
  1. small - от 0 до 767px
  2. medium - от 768px до 991px
  3. large - от 992px до 1199px
  4. xlarge - от 1200px до 1599px
  5. xxlarge - от 1600px

Использование:

flexy-breakpoint

@include flexy-breakpoint(large, print);
.b-breakpoint {
  padding: 10px;
  text-align: center;

  @include flexy-breakpoint(small) {
    background-color: gray;
  }

  @include flexy-breakpoint(medium) {
    background-color: green;
  }

  @include flexy-breakpoint(large) {
    background-color: blue;
  }

  @include flexy-breakpoint(xlarge) {
    background-color: yellow;
  }

  @include flexy-breakpoint(xxlarge) {
    background-color: orange;
  }
}

Результат

.b-breakpoint {
  padding: 10px;
  text-align: center;
}

@media screen and (min-width: 0) {
  .b-breakpoint {
    background-color: gray;
  }
}

@media screen and (min-width: 768px) {
  .b-breakpoint {
    background-color: green;
  }
}

@media screen and (min-width: 992px) {
  .b-breakpoint {
    background-color: blue;
  }
}

@media screen and (min-width: 1200px) {
  .b-breakpoint {
    background-color: yellow;
  }
}

@media screen and (min-width: 1600px) {
  .b-breakpoint {
    background-color: orange;
  }
}

flexy-media

@include flexy-media((min-width: 600px, max-width: 700px), print);

Код:

.b-media {
  @include flexy-media((min-width: 600px, max-width: 700px)) {
    background-color: purple;
  }
}

Результат:

@media screen and (min-width: 600px) and (max-width: 700px) {
  .b-media {
    background-color: purple;
  }
}

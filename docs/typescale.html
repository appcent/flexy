Описание

Миксины для использования Modular Scale

Использование

Код:

@include flexy-type-scale(16px, flexy-modular-scale(perfect-fourth));

Результат:

h6 {
  font-size: 0.8002em;
  line-height: 16px;
}

h5 {
  font-size: 1.06667em;
  line-height: 21.328px;
}

h4 {
  font-size: 1.42187em;
  line-height: 28.43022px;
}

h3 {
  font-size: 1.89535em;
  line-height: 37.89749px;
}

h2 {
  font-size: 2.5265em;
  line-height: 50.51735px;
}

h1 {
  font-size: 3.36782em;
  line-height: 67.33963px;
}

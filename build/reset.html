Reset

`reset` состоит из 2 частей.

_reset_base.scss предназначен для сброса общих стилей страницы.

_reset_form.scss предназначен для сброса стилей form, input, button, textarea, select и всевдо элементов ::placeholder. Так же убирает outline.
